# Legacy Scripts Archive

This directory contains the original standalone scripts and development files that were used before the refactored architecture.

## Files

- `logitech_g560.py` - Original standalone G560 control script
- `test_all_zones.py` - Test script for all zones
- `test_claim_g560.py` - USB device claim test
- `test_one_zone_red.py` - Single zone test
- `testing.ipynb` - Development notebook

## Usage

These files are preserved for reference but are no longer maintained. The functionality has been integrated into the main `logitech_lighting` package with improved architecture.

For current usage, see the main README and use:
- `logitech-lighting` CLI
- `g560ctl` CLI (backward compatible)
